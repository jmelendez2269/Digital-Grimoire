Contains documentation related to AWS S3 setup and configuration. These documents are obsolete as the project has migrated to Cloudflare R2 for object storage.

**Files:**
- `guides/UPLOAD_FIX_QUICK_GUIDE.md` - Quick fix guide for AWS S3 CORS configuration
- `Setup Docs/AWS_S3_SETUP.md` - AWS S3 setup guide

**Reason for Archive:** System migrated from AWS S3 to Cloudflare R2 in October 2025.
