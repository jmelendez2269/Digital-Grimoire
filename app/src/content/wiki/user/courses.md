# User Guide: Prismatic Learning (Courses)

> [!NOTE]
> The Courses surface is now branded **Prismatic Learning**. The route remains `/courses`. Older planning docs may still refer to `Project Parallax Academy` or `Parallax School`.

## Overview

**Prismatic Learning** is Prismarium's structured learning experience. Where the Library invites open exploration, Prismatic Learning curates guided journeys through specific traditions, concepts, and practices.

The page is organised around four tabs:

| Tab | What it is |
|---|---|
| **Catalog** | The full list of individual courses you can start today. Each card shows the title, difficulty, estimated time, and tier (free vs. paid). |
| **Arcs** | Multi-course **arcs** that group several courses around a single throughline (for example, a tradition or a recurring question). Treat an arc as a short curriculum. |
| **Paths** | Longer **paths** that thread several arcs and courses into a months-long study plan. Paths are the deepest formal experience the platform offers. |
| **Map** | A visual map of how courses, arcs, and paths connect, including prerequisites and recommended next steps. |

## How It Works

1. **Pick a starting surface.** Browse the **Catalog** to grab a single course, or open **Arcs** / **Paths** if you want a curated sequence. Use **Map** when you want to see the whole landscape before committing.
2. **Open a course.** Each course detail page shows its modules, readings, and what kind of work each module asks for.
3. **Move through the modules.** Modules combine:
   * **Readings** — Curated excerpts from the Library, each with a long-form **[Reading Digest](/wiki/reading-digests)** that summarises the source.
   * **Video / Audio** — Multimedia content where appropriate.
   * **Quizzes** — Knowledge checks to confirm retention.
   * **Practical Exercises** — Rituals, meditations, or journal prompts that move the work off the page.
4. **Track progress.** Progress saves automatically. You can pause and resume any course, arc, or path at any time.

## Free vs. Paid Courses

Some courses are open to all users; others are gated behind a Premium subscription. Course cards mark paid courses clearly, and the course detail page will prompt you to upgrade if a paid course is selected without an active subscription. Free previews are available for some paid courses.

## Types of Offerings

* **Introductory courses** — "Esotericism 101", "Tarot Basics", "Meditation Fundamentals".
* **Tradition-specific courses** — "Hermeticism", "Kabbalah", "Alchemy".
* **Thematic arcs** — "The Hero's Journey", "Shadow Work", "Sacred Geometry".
* **Multi-month paths** — Curated sequences that build from foundational material into focused practice.

## Certification

Completing certain advanced courses or full paths earns a **digital badge** displayed on your user profile.
